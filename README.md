# Search-ExchangeProxyNotShell
 Search for Exchange Zero-Day ProxyNotShell IOCs

## Usage
Download and copy this script to an Excchange Server.
Run this script:

```
.\Search-ExchangeProxyNotShell.ps1
```

 - Windows Server 2022
